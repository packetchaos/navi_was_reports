## Directions

### Install Requirements
    python3 >= 3.6
    pip3 install requests
    pip3 install flask

### Clone the Repository
    git clone https://github.com/packetchaos/navi_was_reports.git

### Add your network keys.
    nano api_wrapper.py

### Run the was scan script
    python3 ./was_report_gen.py
